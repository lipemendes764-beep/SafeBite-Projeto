# SafeBite

CURSO: Análise e Desenvolvimento de Sistemas

DISCIPLINA: Trabalho Interdisciplinar Aplicações Web Front-End

1º semestre/2026

O projeto tem como objetivo desenvolver uma plataforma digital voltada para auxiliar pessoas com restrições alimentares a encontrarem restaurantes que ofereçam opções seguras e adequadas às suas necessidades. A solução busca centralizar informações sobre ingredientes, preparo dos alimentos e possíveis riscos, permitindo que o usuário tome decisões mais conscientes e seguras ao escolher suas refeições e estabelecimentos.

Além disso, o sistema pretende promover a inclusão alimentar ao dar visibilidade a estabelecimentos que se preocupam com esse público, contribuindo para uma melhor experiência do usuário e também a redução de riscos à saúde.
